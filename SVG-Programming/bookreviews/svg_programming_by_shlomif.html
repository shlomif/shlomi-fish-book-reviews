SVG Programming : The Graphical Web

Title: SVG Programming : The Graphical Web
Author: Kurt Cagle
Publisher Apress
ISBN: 1-59059-019-8
Publishing date: July 2003
Pages: 586
Reviewer: Shlomi Fish
Review date: July 2003
Rating:

I received this book as a present from the publisher in the latest Perl conference in Israel. After reading it almost completely, I can say, that I think very highly of it. This book provides a good introduction to SVG (Scalable Vector Graphics), and I believe will also serve as a handy reference.

The author's style is simple and captivating, and I enjoyed reading the book very much. The book is annonated with many examples that contain code and output, so it demonstrates the concepts that are introduced very well too.

All in all, a great buy, if you're interested in learning how to write SVG images.
